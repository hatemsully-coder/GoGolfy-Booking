QR codes only work once the app lives at a public web address. Here is the fastest free way:
Option A — Netlify Drop (no account, ~3 min)
  1. Open app.netlify.com/drop on a computer.
  2. Drag the whole GoGolfy folder onto the page.
  3. You instantly get a URL like https://random-name.netlify.app/.
  4. Copy it → paste into the QR Codes tab → Generate → Print.
  5. (Free Netlify sign-up keeps the site permanent.)
Option B — GitHub Pages (free, permanent)
  1. Create a free github.com account + new repository.
  2. Upload index.html.
  3. Settings → Pages → deploy from main branch → root.
  4. Your URL becomes https://USERNAME.github.io/REPO/.
Tip: after hosting, open your live URL once and set the driver's WhatsApp number in Settings on the device you'll use to manage bookings.
